# Google Scholar MCP v0.1.1

## Highlights

- Added a more complete Go MCP server surface with `resources` and `prompts`.
- Added CLI commands for `stdio`, `version`, and `help`.
- Added CI, Docker packaging, release build automation, and installation docs.
- Added Homebrew and Scoop install definitions backed by GitHub Release archives.

## Validation

- `GOLANG_PROTOBUF_REGISTRATION_CONFLICT=warn go test ./...`
- `./scripts/verify_stdio.sh smoke`

## Install

### Go

```bash
go install github.com/bingshuoguo/google-scholar-mcp/cmd/google-scholar-mcp@v0.1.1
```

### Homebrew

```bash
brew install --formula https://raw.githubusercontent.com/bingshuoguo/google-scholar-mcp/main/packaging/homebrew/google-scholar-mcp.rb
```

### Scoop

```powershell
scoop install https://raw.githubusercontent.com/bingshuoguo/google-scholar-mcp/main/packaging/scoop/google-scholar-mcp.json
```
