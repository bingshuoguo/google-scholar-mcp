# Changelog

All notable changes to this project will be documented in this file.

## [v0.1.1] - 2026-03-17

### Added

- Added richer MCP server surface with resources and prompts alongside tools.
- Added CLI subcommands for `stdio`, `version`, and `help`.
- Added `CHANGELOG.md`, release build script, and installation documentation.
- Added Homebrew formula and Scoop manifest definitions backed by GitHub Release assets.
- Added GitHub Actions CI and Docker packaging.

### Changed

- Expanded the stdio smoke test to validate tools, resources, resource templates, and prompts.
- Improved README and Chinese README with release-based installation paths.

## [v0.1.0] - 2026-03-17

### Added

- Initial Go-based Google Scholar MCP server release.
- Added keyword search, advanced search, and author profile tools.
- Added parser fixture tests and local stdio smoke validation.
- Added Codex, Cursor, Claude, and Gemini client integration guides.
